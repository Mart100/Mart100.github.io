Tutorial: Setting up a tournament


1: Creating a brand new tournament

You can create a new tournament using =create [Tournament name]


2: Setting up for players to join

There are two ways for players to join your tournament:

The first way for players to join your tournament is with this command: =join [Tournament name]

The second way is to make it possible for players to join with a reaction to a message.
To make this possible, first send a message which the reaction will be binded to.
For example, "React with ✅ To join the new 1v1 rocket league tournament. Tournament starts on 20 Sept"
Then, copy the ID of that message (If you don't know how to get a message ID, follow this tutorial)
and use this command: =tournament [Tournament name] reactmessage [message ID]
If done correctly, the bot will automatically react to the message with a ✅.
Then you're set! Players will automatically join when reacting to that message!


3: Starting the tournament

Starting the tournament means that players will no longer be able to join
and that players will be able to complete their matches.
Starting a match is as simple as =tournament [Tournament name] start


4: Results

The great thing about using Toorney is that you, as the Admin, don't have to input all the results manually.
Players can do this themselves with this command: =results [Tournament name] [player score] [opponent score]
If there are any issues with players using this command, Admins can always force results using:
=forceresults [Tournament name] [player1] [player2] [player1 score] [player2 score]
