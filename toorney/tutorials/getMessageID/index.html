Tutorial: Get a message ID


1: Enabling developer mode

First thing you need to do is make sure you have Discord Developer Mode enabled.
To do this, go to your Discord settings, go to Appearance, then at the bottom, enable Developer Mode.


2: Getting the message ID

Right click the message ID you want. And click "Copy ID"


That's it, now you have that message ID copied to your clipboard!
